# Professional Portfolio (Frontend + Node.js Backend)

This project is a modern personal portfolio with:

- A professional two-color UI theme (slate + cyan)
- React frontend
- Node.js + Express backend APIs
- `Next.js` included in dependencies for your professional stack alignment

## Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Additional: Next.js, REST API architecture

## Scripts

From the `alice-portfolio` directory:

- `npm run dev` - Run frontend and backend together
- `npm start` - Run frontend only
- `npm run backend` - Run backend only
- `npm run build` - Build frontend for production
- `npm test` - Run tests

## API Endpoints

- `GET /api/health` - Backend health check
- `GET /api/profile` - Portfolio profile data for hero section
- `POST /api/contact` - Contact form submission endpoint

## Quick Start

1. Install dependencies:

   `npm install`

2. Start both frontend and backend:

   `npm run dev`

3. Open:

   [http://localhost:3000](http://localhost:3000)
